Built CloudScan!

An asynchronous cloud intelligence platform that scans AWS infrastructure, analyzes CloudWatch metrics, estimates cloud cost, tracks historical trends, and assigns explainable risk scores to identify underutilized resources.

CloudFriend3 combines queue workers, schedulers, Slack alerts, PostgreSQL persistence, and API reporting to transform cloud visibility into actionable engineering insights.

Built using Python, AWS, Redis, PostgreSQL, FastAPI, CloudWatch, and APScheduler.



Repo: https://github.com/Mannojj18/cloudscan



Workflow:



Scedular ->Trigger → Queue → Worker → AWS Scan → Metrics → Cost → Trend → Risk → Storage → Slack Alerts → Reports
